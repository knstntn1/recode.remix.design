# Plan 01-03: Foundation Verification and Visual Checks

## Objective

Add automated verification for the Phase 1 shell: production build, static output checks, and Playwright desktop/mobile rendering checks for the Computingeducation-inspired placeholder site.

## Context

Read before execution:

- `.planning/phases/01-static-site-foundation-and-visual-system/01-CONTEXT.md`
- `.planning/phases/01-static-site-foundation-and-visual-system/01-UI-SPEC.md`
- `.planning/REQUIREMENTS.md`
- `package.json`
- `src/layouts/BaseLayout.astro`
- `src/components/SiteHeader.astro`
- `src/components/SiteFooter.astro`
- `src/styles/global.css`
- `src/pages/index.astro`
- `src/pages/exponate/[slug].astro`

## Threat Model

<threat_model>
**Scope:** Verification tooling for local static build output.

**Threats considered:**
- False confidence from build-only verification while visual shell breaks on mobile.
- Tests accidentally requiring remote source media or live Computingeducation assets.
- Generated screenshots or build output being committed accidentally.

**Mitigations:**
- Add Playwright checks at desktop and mobile viewports.
- Assert expected local text/CSS contracts rather than remote assets.
- Add generated output directories to `.gitignore`.

**Blocking severity:** A verification setup that passes without checking mobile rendering is blocking for this plan.
</threat_model>

<action>
Create `tests/phase1-shell.spec.mjs` as a Playwright script using `chromium` that satisfies D-08 and D-09:
- starts from a base URL passed as `process.env.PLAYWRIGHT_BASE_URL || 'http://127.0.0.1:4321'`
- checks `/` at viewport `1280x900`
- checks `/` at viewport `390x844`
- checks `/exponate/platzhalter/` at viewport `390x844`
- asserts visible text `Recode.Remix.Design`
- asserts visible text `ComputingEducation 2026`
- asserts the Impressum link has href `https://computingeducation.de/impressum/`
- evaluates `document.documentElement.scrollWidth <= document.documentElement.clientWidth` at each viewport
- writes screenshots to `.planning/artifacts/phase-01/desktop-home.png`, `.planning/artifacts/phase-01/mobile-home.png`, and `.planning/artifacts/phase-01/mobile-exhibit.png`

Add package script:
- `"test:visual": "npm run build && (npm run preview -- --host 127.0.0.1 > /tmp/recode-preview.log 2>&1 & PREVIEW_PID=$!; sleep 3; PLAYWRIGHT_BASE_URL=http://127.0.0.1:4321 node tests/phase1-shell.spec.mjs; STATUS=$?; kill $PREVIEW_PID; exit $STATUS)"`
</action>
