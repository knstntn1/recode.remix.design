# Plan 02-03: Validate Migration Completeness and Independence

## Objective

Add migration-specific verification so later phases can rely on local content and assets without re-checking the source migration manually.

## Tasks

<task>
<name>Add migration verification script</name>
<files>
- `scripts/verify-migration.mjs`
- `package.json`
</files>
<action>
Create `scripts/verify-migration.mjs` that exits non-zero unless:
- `src/data/exhibit-manifest.json` exists and has exactly 17 exhibits
- `src/data/exhibition.json` exists and has non-empty `contentHtml`
- every manifest exhibit has `src/data/exhibits/{slug}.json`
- every exhibit JSON has non-empty `contentHtml`
- every media local path points to an existing file under `public/`
- no file under `src/data` contains `experimente.joachim-wedekind.de/wp-content`
- `500 Linien` has slug `500-linien` and source page ID `625`

Add package script:
- `"verify:migration": "node scripts/verify-migration.mjs"`
</action>

</task>
